### 我们总想改变命运，却从未看清那编织命运的手——我们自己的心智。  
#### 本书最大亮点  
完整呈现一套“根植于禅宗智慧、融合现代认知科学、极具开创性”的「心智论」模型：  
-	一心：觉性本体，生命源头。  
-	二机制：注意力与分别心的迷失与唤醒。  
-	三门：决定命运的瞬间抉择。  
-	三网：从默认网络到觉性网络的境界提升。  
#### 清晰的旅程路径  
基于这张心智地图，本书将带你走过环环相扣的六部觉醒归程：  
> *困局* · 识得庐山真面目 → *地图* · 云在青天水在瓶 → *训练* · 功夫在诗外 → *境界* · 也无风雨也无晴 → *创造* · 从自主到自在 → *圆满* · 归途如歌  
感恩与祝愿  


生逢中国盛世，国泰民安，我们得以安居乐业，安心向内，探寻生命的根本智慧。值此文化复兴的伟大时代，对内在心性的深刻探索，正是我们建立深层文化自信的基石。这趟“从知道到体证”的觉醒之旅，正是植根于中华优秀传统文化的一次内在实践。  
愿你在此，识取本心，拿回生命的主动权，活出内心的宁静与自在，并将这份觉醒的光明，回馈于我们深爱的、正致力于民族复兴的伟大时代与国土。
